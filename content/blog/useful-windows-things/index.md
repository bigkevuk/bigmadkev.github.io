Some useful Windows things that I always forget

<!--more-->
## Variables and Shortcuts

### Startup folder
```start > run > shell:startup```
### Widows Store Apps folder
```start > run > shell:appsfolder```
### Roaming Appdata
```%APPDATA%```
### Program Files
```%ProgramFiles%```
### Userprofile
```%USERPROFILE%```

Can find more at https://ss64.com/nt/syntax-variables.html
